Powerful Features

Everything you need in a modern web server

Non-Blocking I/O

Event-driven architecture using epoll for handling thousands of concurrent connections efficiently.

CGI Support

Execute dynamic scripts with full CGI/1.1 support for Python, PHP, Ruby, and shell scripts.

File Uploads

Handle large file uploads with configurable size limits and multipart/form-data support.

HTTP/1.0 Protocol

Full HTTP/1.0 implementation with GET, POST, DELETE methods and proper status codes.

Custom Configuration

nginx-inspired configuration format with locations and error pages.

Custom Error Pages

Configurable error pages with automatic fallback to built-in defaults for reliability.

Simple Yet Powerful

Get your server running with just a simple configuration file. No complex setup, no unnecessary dependencies.

Zero Dependencies

Pure C++98, uses only standard POSIX system calls

Memory Safe

Carefully managed memory, no leaks, handles errors gracefully

Production Ready

Handles timeouts, keeps connections alive, serves static files efficiently

config/default.conf 
server {
    listen 8080;
    host 127.0.0.1;
    server_name webserv/1.0;
    root www;
	client_max_body_size 3000000;
	index index.html;

	error_page 404 errors/404.html;
	error_page 404 errors/403.html;
	error_page 404 errors/413.html;
	error_page 404 errors/500.html;
	error_page 404 errors/504.html;
    
    location / {
        allowed_methods GET POST;
        index index.html;
    }
    
	cgi_conf .py /usr/bin/python3;

}
